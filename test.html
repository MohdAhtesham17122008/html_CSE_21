Frequently Asked Questions (FAQ)

1. What is HTML?
HTML stands for HyperText Markup Language. It is used to create the structure of a webpage.
2. What is CSS?
CSS stands for Cascading Style Sheets. It is used to style and design HTML elements.
3. What is JavaScript?
JavaScript is a scripting language used to make webpages interactive and dynamic.
4. What is the difference between HTML and CSS?
HTML provides the structure of the webpage, while CSS adds styling and layout.
5. What is a responsive website?
A responsive website adjusts its layout automatically to fit different screen sizes like mobile, tablet, and desktop.
